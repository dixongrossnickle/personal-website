I'm a senior at UT Austin majoring in MIS. I built this website to learn about full stack web development. Click the icons for more information.

Python

Python is the reason I made this site. It was the first language I learned, and I wanted to build an interface for the soccer simulator I wrote. It uses Python to analyze season statistics and FIFA ratings, and then chooses an outcome based on probability distributions.

Django

Django is the web framework that runs the site. I chose to learn Django because not only is it written in Python, it's also very powerful. This site uses only a fraction of Django's functionality â€” I just liked it a lot and wanted to get familiar with it.

JavaScript

Building this website was my first time using JavaScript. At first, I just needed a few functions for the site to work. But now, it practically runs on JavaScript; the simulator uses it to send requests to the server, receive the results as JSON, and update the page.

Bootstrap

This was also my first introduction to HTML and CSS, but working with Bootstrap taught me a lot. It was much simpler to create a responsive layout than I had imagined, and I was able to use Bootstrap's JavaScript events for some nice effects.
