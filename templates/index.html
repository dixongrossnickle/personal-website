languages | frameworks | libraries

Click the icons for more information about this site and my experience.

Python

Python is the reason I built this site. It was the first language I learned, and I wanted to build an interface for a program I had written. The simulator uses Python to analyze season statistics and FIFA ratings, and then chooses an outcome based on probability distributions.

Django

Django is the web framework that runs the site. I chose to learn Django because not only is it written in Python, it's also very powerful. This site uses only a fraction of Django's functionality — I just liked it a lot and wanted to get familiar with it.

JavaScript

Building this website was my first time using JavaScript. At first, I just needed a few functions for the site to work. But now, it practically runs on JavaScript; the simulator uses it to send requests to the server, receive the results as JSON, and update the page.

HTML/CSS

This was also my first introduction to HTML and CSS, but working with Bootstrap taught me a lot. It was much simpler to create a responsive layout than I imagined, and two components were all I needed for the simulator.
